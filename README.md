## 📄 Research & Publications

### 🛰️ Domain-Adapted Video Frame Interpolation for Atmospheric Satellite Imagery — *IC2NC 2025 (IEEE)*
Optical-flow-weighted loss function modeling non-rigid cloud dynamics to eliminate artifacts in standard Video Frame Interpolation. Achieved **+1.67 dB PSNR (32.75 → 34.42)** and **SSIM 0.8737 → 0.8991** via transfer learning (Vimeo-90K → WMS) on a **94 GB** dataset, then deployed as a GPU-accelerated SaaS inference platform with REST APIs and a web UI.
> `PyTorch` · `CUDA` · `FastAPI` · `React` · `Docker`
>
> 📑 [Paper (IEEE)](https://ieeexplore.ieee.org/document/11376065) · 💻 [Code: WMS-RIFEv3](https://github.com/Anson-Saju-George/wms-rifev3) · 🗂️ [Dataset (94 GB)](https://huggingface.co/datasets/Anson-Saju-George/wms_dataset) · 🌐 [Live Demo](https://ansonsajugeorge.online/wms/)

### 🕵️ Protocol-Separated Deepfake Detection Across Modalities — *WIP Research*
Image & video deepfake detection benchmarking **ConvNeXt, ViT, Swin, TCN, LSTM, ConvLSTM** with a protocol-separated, identity-aware evaluation framework to prevent leakage. **98.20% image accuracy**; a **TCN** achieved the strongest video performance (**91.02% accuracy, 0.7831 F1**). Conducting research for publication on spatial vs. temporal learning for multimedia forensics.
> `PyTorch` · `ConvNeXt` · `ViT` · `Swin` · `TCN` · `OpenCV` · `Scikit-learn`
>
> 💻 [Research Repo](https://github.com/Anson-Saju-George/Deepfake-Engine-Research) · 🌐 [Live Demo](https://ansonsajugeorge.online/deepfake/)

### 🔐 Deep Learning-Based Network Traffic Classification (CNN + RNN) — *IEEE*
Hybrid CNN-RNN architecture classifying encrypted/non-encrypted network traffic via spatial packet features and temporal sequence modeling — **96.4% accuracy, 0.94 F1**.
> 📑 [Paper (IEEE)](https://ieeexplore.ieee.org/abstract/document/10125858)

## 🛠️ Featured Projects

### 🔎 [Hybrid RAG System — ContextForge](https://ansonsajugeorge.online/context-forge/)
Local-first RAG workbench comparing versioned retrieval pipelines (v0 → v3.1) over a fixed corpus. Hybrid retrieval (BM25 + embeddings + reranking), citation tracing, tool calling, structured outputs, and agentic multi-turn orchestration. Warm-query latency **~150–500 ms** with retrieval caching; production access control via Google OAuth + Razorpay + SQLite usage tracking.
> `FastAPI` · `React` · `SQLite` · `BM25` · `Ollama` · `Qwen3` · `Hybrid Retrieval` · `Tool Calling` · `OAuth` · `Razorpay`
>
> 🌐 [Live Demo](https://ansonsajugeorge.online/context-forge/)

### 🖥️ [Self-Hosted AI Infrastructure & GPU-ML Home Server](https://github.com/Anson-Saju-George/home_server)
Self-hosted AI platform on Fedora Linux (RTX 3060) using Docker, Nginx, Cloudflare Tunnel, and Tailscale for secure deployment and remote access. Hosts CUDA-enabled FastAPI inference services, RAG systems, CV services, and this portfolio — with SSH hardening, firewall management, automated backups, and systemd-based multi-app deployment automation.
> `Fedora` · `Docker` · `Nginx` · `CUDA` · `Cloudflare Tunnel` · `Tailscale` · `FastAPI` · `systemd` · `UFW`

### 🧬 [Deepfake Detection System — DF-Engine](https://ansonsajugeorge.online/deepfake/)
Full-stack ML platform with separate image/video inference pipelines (ConvNeXt, ViT, Swin, MaxViT). **98.69% image F1** (ConvNeXt-Base); **88.68% video accuracy** (ConvNeXt-Large temporal). FastAPI inference with GPU memory-aware scheduling, model caching, hashed job queues, and VRAM offloading.
