## 📄 Publications

### 🛰️ Domain-Adapted Video Frame Interpolation for Atmospheric Satellite Imagery — *IC2NC 2025 (IEEE) · Lead Author*
*Anson Saju George, T. Jemima Jebaseeli.* Fine-tuned a RIFE-based model to adapt generic video frame interpolation to atmospheric satellite imagery via transfer learning (Vimeo-90K / X4K1000FPS → WMS). A custom cloud-aware **motion-weighted loss** (L1 + motion-weighted + SSIM + smoothness) handles non-rigid cloud dynamics, improving **PSNR 32.75 → 34.42 dB (+1.67 dB)** and **SSIM 0.8739 → 0.8991**. Curated and open-sourced a **94+ GB** VIIRS/NOAA cloud-motion dataset and the trained model, then deployed a GPU-accelerated web platform with a job-queue inference pipeline.
> `PyTorch` · `RIFE` · `CUDA` · `FastAPI` · `React` · `Docker` · `Nginx`
>
> 📑 [Paper (IEEE)](https://ieeexplore.ieee.org/document/11376065) · 🤗 [Hugging Face](https://huggingface.co/Anson-Saju-George) · 💻 [Code: WMS-RIFEv3](https://github.com/Anson-Saju-George/wms-rifev3) · 🗂️ [Dataset (94 GB)](https://huggingface.co/datasets/Anson-Saju-George/wms_dataset) · 🌐 [Live Demo](https://ansonsajugeorge.online/wms/)

### 🔐 A Robust Deep Learning-based Approach for Network Traffic Classification using CNNs and RNNs — *ICSPC 2023 (IEEE) · Co-Author*

## 🛠️ Featured Projects

### 🧬 [DF-Engine — Deepfake Detection Research Pipeline (Image & Video)](https://ansonsajugeorge.online/deepfake/)
Research-grade, protocol-aware deepfake detection over a **192K-sample corpus** (179,988 images + 12,028 videos) with **identity-aware splitting** to prevent subject leakage, across **22 tracked experiment runs**. Benchmarked image backbones (ViT, ConvNeXt, Swin, DeiT, ConvNeXtV2, MaxViT, EVA) — best image model **ConvNeXt-Base: F1 0.9863, ROC-AUC 0.9968**. Compared temporal/spatiotemporal video heads (mean, LSTM, Transformer, TCN, ConvLSTM, hybrids) — best video model **91.0% accuracy / 0.784 F1**, with **ConvNeXt-TCN** strongest. FastAPI inference with VRAM-aware scheduling, model caching, and GPU memory offloading.
> `PyTorch` · `timm` · `ConvNeXt` · `ViT` · `Swin` · `TCN` · `OpenCV` · `FastAPI`
>
> 🌐 [Live Demo](https://ansonsajugeorge.online/deepfake/) · 💻 [Code](https://github.com/Anson-Saju-George/df-engine) · 🔬 [Research Repo](https://github.com/Anson-Saju-George/Deepfake-Engine-Research)

### 🔎 [ContextForge — Config-Aware RAG Retrieval Workbench](https://ansonsajugeorge.online/context-forge/)
Local-first RAG workbench comparing retrieval architectures (v0 → v3.1) over a fixed corpus, with citation tracing, retrieval telemetry, and benchmark quality gates for evidence-grounded answers. Engineered a **hybrid retrieval stack from scratch** — BM25 sparse retrieval, query intent routing, document-balanced synthesis, deterministic specificity reranking, MMR-style diversity, and parent-child context expansion — no external framework. Reduced warmed-query latency to **~160–200 ms** for synthesis (sub-second extraction) on a **9,706-chunk** corpus. Google OAuth, Razorpay hourly access, and server-side SQLite usage tracking.
> `FastAPI` · `React` · `SQLite` · `BM25` · `Ollama (Qwen3 / Gemma)` · `Hybrid Retrieval` · `OAuth` · `Razorpay`
>
> 🌐 [Live Demo](https://ansonsajugeorge.online/context-forge/)

### 🖥️ [Self-Hosted AI Infrastructure (Home Server)](https://github.com/Anson-Saju-George/home_server)
GPU-enabled self-hosted platform on Fedora Linux (repurposed hardware, RTX 3060) serving multiple AI apps — WMS, deepfake detection, ContextForge, EEG demo, and portfolio — behind a single hardened gateway. Nginx reverse proxy routes per-port FastAPI services under one domain via systemd, with **Cloudflare Tunnel** (zero-trust TLS) and **Tailscale** mesh VPN. Hardened with key-only SSH on a non-standard port, UFW, Fail2Ban, automated security updates, and scheduled backups; runs CUDA inference services and self-hosted Nextcloud.
> `Fedora` · `Docker` · `Nginx` · `CUDA` · `Cloudflare Tunnel` · `Tailscale` · `systemd` · `UFW` · `Fail2Ban`
>
> 💻 [Code](https://github.com/Anson-Saju-George/home_server)
